# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of AuriPlan
- 2D floor plan editor with wall drawing tools
- 3D visualization with Three.js
- Furniture catalog with 8000+ items
- AI design assistant
- Real-time collaboration
- Export to multiple formats (PNG, PDF, OBJ, SVG, JSON)
- User authentication and authorization
- Project management dashboard
- Multi-language support (PT, EN, ES, FR, DE)
- Responsive design

## [1.0.0] - 2024-01-01

### Added
- Core editor functionality
- Wall, room, door, and window tools
- Furniture library with categories
- Material and color customization
- Undo/redo system
- Snap system for precision
- Grid system
- Camera controls for 2D and 3D views
- Export functionality
- User authentication
- Project persistence
- Collaboration features
- WebSocket real-time updates
- Dashboard interface
- Settings and preferences

### Security
- JWT-based authentication
- Password hashing with bcrypt
- Input validation
- XSS protection
- CSRF protection
